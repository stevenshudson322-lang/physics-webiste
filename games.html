Cannon Launching Simulator

Welcome to the Cannon Launching Simulator! This interactive physics simulation lets you practice projectile motion calculations.

How to play:

  • Press SPACE to start a new round
  • Enter an angle (0-90 degrees)
  • Enter a velocity (1-50 m/s)
  • Watch the trajectory and try to hit the target!
  • Use the measurement scales to help calculate the correct angle and velocity

To run the game locally:

  1. Download the Python file: cannonSim.py
  2. Make sure you have Python 3 installed
  3. Install pygame: pip install pygame
  4. Run the game: python cannonSim.py
